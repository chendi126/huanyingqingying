## 🚀 如何使用

### 启动开发服务器
```bash
cd huapi-website
npm run dev
```

### 访问网站
打开浏览器访问：`http://localhost:5173/`

### 运行测试
```bash
npm test
```

### 构建生产版本
```bash
npm run build
```
